Facebook Event Creator Help

First of all, sorry your on this page because it means that somethings probably not working like you want :) No worries, we'll get it sorted out. Tip for you: Hover over the large questions marks in the app for quick help. If you click on them they will disappear!

Prerequisites

You need to have some basic stuff before can even start using this application.

  • A facebook account
  • A facebook developer account account
  • A facebook event page

Facebook Application Stuff

If you created your Facebook developers account for the sole purpose of using it with this application and you don't know "how your app integrates with Facebook", make sure you select "Website with Facebook Login". You can put any value in there that you want as long as it starts with http://... so just put something like http://myapp.com and you're done.

Excel Notes

Right now, only two columns are supported in Excel:

  • Column 1: The event name
  • Column 2: The event start date

I would recommend that you set the cells with the date / time values for the events to Date format in excel. You don't have to, but this way if you enter the date and time it will be set correctly.
